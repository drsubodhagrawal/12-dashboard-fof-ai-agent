How Each Phone Type is Handled
📱 Mobile
SMS capability confirmed. Aria sends the mid-call SMS fallback text when STT confidence drops below the threshold.
→ Layer 2 SMS active
📟 Landline
Cannot receive SMS. Aria skips Layer 2 entirely and jumps to Layer 3 graceful skip after 2 voice attempts. No failed SMS delivery.
→ Layer 3 graceful skip
💻 VoIP (Fixed)
Many VoIP numbers support SMS. Aria attempts the text; if delivery fails (Twilio error 21614), automatically falls back to Layer 3.
→ Attempt SMS, catch error
❓ Unknown
Lookup returned unknown type. Aria attempts SMS conservatively; catches any failure gracefully at the Twilio error level.
→ Attempt SMS, catch error
